# qrcode-generator

A simple QR code generator built with React + Vite + Tailwind CSS.

## Features
- Generate QR codes from text or URLs
- Customizable size and error correction level
- Foreground and background color options
- Logo upload support
- Download as PNG or SVG

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output is in `dist/` — deploy to Cloudflare Pages or any static hosting.
